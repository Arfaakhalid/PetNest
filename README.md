PetNest Network is a role-based full stack web application designed to handle pet adoption, rescue operations, and animal-related commerce within a single, unified platform.
The system is built to reflect real-world workflows, enforce strict role boundaries, and maintain accountability across all user actions.

This project is engineered as a production-grade application rather than a tutorial or academic demo.

## Key Goals

- Centralize pet adoption, rescue, and animal trading
- Enforce role-based access and permissions
- Enable real-time communication between stakeholders
- Maintain data integrity and operational transparency
- Support scalable and maintainable system growth

## User Roles
## Admin:
- Manage and verify all users and roles
- Approve, edit, or remove rescue organizations
- Monitor and control all reported animal cases
- Update case statuses (pending, under work, resolved)
- Moderate testimonials and platform content
- Maintain secure authentication and access control
- Secure signup and login

## Reporter:
- Submit detailed reports for stray, sick, or deceased animals
- Upload mandatory images or videos with location data
- View report history and real-time case progress
- Manage personal profile and identity details
- Earn Star Reporter status after consistent contributions
- Access rescue organization details and performance metrics

## Adopter
- Secure account access
- Browse animals with advanced filters:
    -Location
    -Animal type and breed
    -Upload date
    -Seller reputation
- View detailed animal profiles (health, habits, media)
- Save, star, and favorite animals
- Initiate adoption workflows
- Real-time chat with sellers (text and media)
- Manage adoption history and preferences
- Access pet accessories marketplace

## Individual Seller
- Secure authentication and profile management
- List animals for sale or adoption with complete metadata
- Upload images, health details, habits, and pricing
- Edit, manage, and close listings
- Track engagement and interest analytics
- Communicate with adopters via real-time chat
- Earn Pro Seller badge after successful deals
- Access pet accessories marketplace

## Shelter / Rescue Organization
- Admin-approved onboarding only
- Access categorized reports:
    -Stray animals
    -Sick animals
    -Deceased animals
- Apply for rescue, recovery, or settlement cases
- Update rescue status (pending, working, completed)
- View reporter profiles and contact details
- Maintain rescue history and performance analytics
- Manage organization profile and public trust indicators

## Pet Accessories Seller
- Secure account and profile management
- Create and manage an independent e-shop
- List pet accessories, food, toys, and supplies
- Edit, remove, and manage product listings
- Track sales and order history
